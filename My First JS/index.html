Some GIT commands

  • $ git touch (file name) - Creates files
  • $ git init - Initialize local repository
  • $ git add . - Add all to index
  • $ git add (file name) - Add spcific file to index
  • $ git rm --cached (file name) - remove from index
  • $ git status - Check status of working tree
  • $ git commit - Commit changes in index
    • Example: $ git commit - Load prompt to add comments
      • Type "I" to enter edit
      • Press "ESC" to exit edit
      • Type ":wq" to commit changes
      • Type ":exit" to exit commit terminal
    • Example: $ git commit -m 'insert comments here'
  • $ git push - Push to remote repository
    • example: $ git push -u origin2 master
  • $ git pull - Pull latest from remote repository
  • $ git clone - Clone repository into a new directory
  • $ git branch (branch name) - Creates a new branch
  • $ git checkout (branch name) - Checks out to the newly created branch
  • $ git merge (from branch) - Merge changes into master branch
    • Type "I" to enter edit
    • Press "ESC" to exit edit
    • Type ":wq" to commit changes
    • Type ":exit" to exit commit terminal
